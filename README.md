# Python Calculator
Python CLI calculator featuring modular architecture, robust error handling, and recent results history.


## Features
- Basic operations (+, -, *, /, %, //, ^)
- Error handling (invalid input, division by zero)
- Modular function-based design
- History tracking (last 5 results)

## How to Run

1. Install Python
2. Run the program:
   python calculator.py

## Example

Enter operation (+, -, *, /, %, ^, //) or 'exit': +
Enter first number: 5
Enter second number: 3
Result: 8
